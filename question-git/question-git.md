#### 3. Git合并分支
git branch：查看分支，当前分支前带*
git checkout branch_name 切换分支
git merge branch_name 将分支合并到master
git add file_name 添加修改的文件
git status 查看分支状态，可以列出冲突的文件
git commit -m "fix merge conflict" 提交合并
